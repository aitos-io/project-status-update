# BoAT Project Status Update
Date: August 25th, 2022

## Update Summary
In the version of 2.2.2, the ECDSA algorithm is optimized and the Fibocom module FM650 is adapted. 86 BoAT core code issues are fixed and optimized, 47 Ethereum unit test cases are fixed,  14 BoAT demo issues are fixed, 5 Chainmaker issues are fixed, 5 cmake issues are fixed, 5 python scripts issues are fixed, 28 warnings are resolved. The supported list file is updated as well as at least 14 descriptions in BoAT documents and README files are updated.


## Releases
[BoAT-X-v2.2.2](https://github.com/aitos-io/BoAT-X-Framework/releases/tag/BoAT-X-v2.2.2) is released.

**Change Log**

- Optimize the ECDSA algorithm
- Adapt to Fibocom module FM650
- Fix 47 Ethereum unit test cases
- Fix 14 BoAT demo issues
- Fix 5 Chainmaker issues
- Fix 5 cmake issues
- Fix 5 python scripts issues
- Resolve 28 warnings
- Fix and optimize 86 BoAT core code issues
- Update the support list file
- Update at least 14 descriptions in BoAT documents and README files
